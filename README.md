## 🚀 Menjalankan Proyek Secara Lokal

### Opsi 1 — Langsung Membuka File

Cukup buka file:

```bash
index.html
```

menggunakan browser.

---

### Opsi 2 — Menggunakan Live Server (VS Code)

1. Install extension **Live Server**
2. Klik kanan pada `index.html`
3. Pilih **Open with Live Server**

---

### Opsi 3 — Python HTTP Server

```bash
python -m http.server 8000
```

Kemudian buka:

```text
http://localhost:8000
```

## 🌐 Deploy ke GitHub Pages

### Push Repository

```bash
git init
git add .
git commit -m "Initial Commit"
git branch -M main
git remote add origin https://github.com/2023-Nizar-186/portfolio.git
git push -u origin main
```

### Aktifkan GitHub Pages

1. Masuk ke repository GitHub
2. Pilih **Settings**
3. Pilih **Pages**
4. Source → **Deploy from a Branch**
5. Branch → **main**
6. Klik **Save**

Website akan tersedia di:

```text
https://2023-nizar-186.github.io/portfolio/
```
